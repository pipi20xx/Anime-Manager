# 识别失败智能重试

当用户询问"为什么这些资源识别失败"、"帮我处理识别失败的条目"时使用此技能。

## 功能

1. **分析失败原因** - 检查标题格式、制作组、TMDB匹配等问题
2. **自动修复** - 尝试使用替代识别策略
3. **提供建议** - 给出用户可操作的建议

## 常见失败原因

| 原因类型 | 特征 | 解决方案 |
|----------|------|----------|
| 未知制作组 | 制作组不在列表中 | 添加制作组到配置 |
| 非标准命名 | 标题格式异常 | 手动指定TMDB ID |
| TMDB无匹配 | 作品未收录 | 使用别名搜索 |
| 季度/集数解析失败 | S01E01格式异常 | 手动指定季度集数 |

## API 端点

| 端点 | 用途 |
|------|------|
| `GET /feeds/{id}/items` | 获取源条目 |
| `POST /rss/recognition/retry` | 全局重试识别 |
| `GET /api/tmdb/search` | TMDB搜索 |
| `POST /api/recognize` | 手动识别 |
