# 番剧/电影发现技能

当用户说"我想看科幻片"时触发此技能。
当用户说"我想看百合番"时触发此技能。
当用户说"推荐一些日常番"时触发此技能。
当用户说"有没有恋爱番"时触发此技能。
当用户说"找热血类型的番"时触发此技能。
当用户说"我想看"时触发此技能。
当用户说"推荐一些"时触发此技能。
当用户说"有没有"时触发此技能。
当用户说"找电影"时触发此技能。
当用户说"有什么电影"时触发此技能。

## 判断媒体类型

- 用户说"番"、"番剧"、"动漫" → 使用 `discover_bangumi`
- 用户说"电影" → 使用 `discover_tmdb` (media_type="movie")
- 用户说"美剧"、"日剧"、"韩剧" → 使用 `discover_tmdb` (media_type="tv")
- 默认 → 使用 `discover_bangumi`

## 执行流程

### 步骤 1: 解析用户意图

从用户输入中提取：
- 媒体类型：番剧/电影/剧集
- 标签/类型
- 年份（如有）

### 步骤 2: 调用工具

**番剧**：调用 `discover_bangumi`
- tag: 提取的标签
- year: 提取的年份
- sort_by: "popularity.desc"

**电影/剧集**：调用 `discover_tmdb`
- media_type: "movie" 或 "tv"
- genre: 提取的类型
- year: 提取的年份
- sort_by: "popularity.desc"

### 步骤 3: 处理结果

工具会返回格式化的表格，直接显示给用户。

如果用户输入编号：
- 番剧 → 调用 `subscribe_by_bangumi_id`
- 电影/剧集 → 调用 `add_subscription`

## 示例

**用户**: 我想看科幻番

**助手**:
1. 媒体类型：番剧
2. 调用 `discover_bangumi(tag="科幻")`
3. 显示科幻番剧列表

---

**用户**: 推荐一些科幻电影

**助手**:
1. 媒体类型：电影
2. 调用 `discover_tmdb(media_type="movie", genre="科幻")`
3. 显示科幻电影列表

---

**用户**: 有没有2024年的百合番

**助手**:
1. 媒体类型：番剧
2. 调用 `discover_bangumi(tag="百合", year="2024")`
3. 显示2024年百合番剧列表
