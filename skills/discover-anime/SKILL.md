# 番剧/电影发现技能

当用户说"我想看科幻片"时触发此技能。
当用户说"我想看百合番"时触发此技能。
当用户说"推荐一些日常番"时触发此技能。
当用户说"有没有恋爱番"时触发此技能。
当用户说"找热血类型的番"时触发此技能。
当用户说"我想看"时触发此技能。
当用户说"推荐一些"时触发此技能。
当用户说"有没有"时触发此技能。
当用户说"找电影"时触发此技能。
当用户说"有什么电影"时触发此技能。

## 重要：上下文保持

**此技能需要两轮对话完成**：
1. 第一轮：显示番剧/电影列表，等待用户选择
2. 第二轮：用户输入数字，执行订阅

## 判断媒体类型

- 用户说"番"、"番剧"、"动漫" → 使用 `discover_bangumi`
- 用户说"电影" → 使用 `discover_tmdb` (media_type="movie")
- 用户说"美剧"、"日剧"、"韩剧" → 使用 `discover_tmdb` (media_type="tv")
- 默认 → 使用 `discover_bangumi`

## 工作流程

### 第一轮：显示列表

从用户输入中提取媒体类型、标签/类型、年份，调用对应工具。

**番剧**：调用 `discover_bangumi(tag=..., year=..., sort_by="popularity.desc")`
**电影/剧集**：调用 `discover_tmdb(media_type=..., genre=..., year=..., sort_by="popularity.desc")`

工具会返回格式化的表格，直接显示给用户。

### 第二轮：处理用户输入

**当用户回复数字时（如 "13" 或 "1 2 3"），你必须直接调用订阅工具！**

**关键**：表格中每一行都包含 Bangumi ID 或 TMDB ID，你需要：
1. 根据用户输入的编号，找到对应行的 ID
2. 番剧 → 直接调用 `subscribe_by_bangumi_id(bangumi_id=xxx)`
3. 电影/剧集 → 直接调用 `add_subscription(title=..., tmdb_id=..., media_type=..., season=1)`

**示例**：用户输入 "13"
- 查看列表第13行，找到 Bangumi ID（例如 604826）
- 调用 `subscribe_by_bangumi_id(bangumi_id=604826)`
- 回复订阅结果

## 禁止事项

1. ❌ 用户输入数字时不要调用 `list_subscriptions`
2. ❌ 不要询问用户想做什么 - 用户输入数字就是要订阅
3. ❌ 不要调用其他不必要的工具

## 示例

**用户**: 我想看科幻番

**助手**:
1. 调用 `discover_bangumi(tag="科幻")`
2. 显示科幻番剧列表

---

**用户**: 13

**助手**:
1. 查看列表第13行，找到 Bangumi ID
2. 调用 `subscribe_by_bangumi_id(bangumi_id=对应ID)`
3. 回复：「✅ 已成功订阅《XXX》」

**用户**: 推荐一些科幻电影

**助手**:
1. 调用 `discover_tmdb(media_type="movie", genre="科幻")`
2. 显示科幻电影列表

---

**用户**: 有没有2024年的百合番

**助手**:
1. 调用 `discover_bangumi(tag="百合", year="2024")`
2. 显示2024年百合番剧列表
