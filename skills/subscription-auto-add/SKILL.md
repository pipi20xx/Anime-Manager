# 本季番剧订阅

当用户说"新番"时触发此技能。
当用户说"本季番剧"时触发此技能。
当用户说"当季番剧"时触发此技能。
当用户说"有什么新番"时触发此技能。

## 重要：上下文保持

**此技能需要两轮对话完成**：
1. 第一轮：显示番剧列表，等待用户选择
2. 第二轮：用户输入数字，执行订阅

**你必须记住第一轮显示的番剧列表，在第二轮根据用户输入的数字执行订阅！**

## 工作流程

### 第一轮：显示番剧列表

1. 调用 `get_bangumi_calendar` 工具获取番剧列表

2. 按以下格式显示，**必须包含 bangumi_id**：

```
📺 本季番剧列表：

| 编号 | 番剧名称 | 更新日 | 评分 | Bangumi ID |
|------|----------|--------|------|------------|
| 1 | 名侦探柯南 | 星期日 | 7.5 | 399 |
| 2 | 航海王 | 星期日 | 8.3 | 975 |
| 3 | Re：从零开始的异世界生活 第四季 | 星期三 | 7.2 | 4765 |
| ...

请输入要订阅的番剧编号（多个用空格分隔，输入"全部"订阅所有）：
```

**重要**：你必须在表格中显示 Bangumi ID，这样你才能在用户选择时知道要订阅哪些！

### 第二轮：处理用户输入

当用户回复数字时（如 "4 6 15" 或 "全部"），**立即执行订阅**：

1. 根据用户输入的编号，找到对应的 Bangumi ID
2. 对每个选中的番剧调用 `subscribe_by_bangumi_id` 工具

**示例**：用户输入 "1 2"
- 第1个番剧的 Bangumi ID 是 399，调用 `subscribe_by_bangumi_id(bangumi_id=399)`
- 第2个番剧的 Bangumi ID 是 975，调用 `subscribe_by_bangumi_id(bangumi_id=975)`

**用户输入 "全部"**：订阅列表中所有番剧

### 返回结果

显示订阅结果：

```
✅ 订阅成功：
- 名侦探柯南
- 航海王

❌ 订阅失败（已存在）：
- 异兽魔都 第二季
```

## 关键提醒

1. **不要询问用户想做什么**，用户输入数字就是想订阅！
2. **记住你显示的番剧列表**，包括编号和 Bangumi ID 的对应关系
3. 用户输入数字后，**直接调用 subscribe_by_bangumi_id 工具**
4. 每订阅一部番剧调用一次工具
