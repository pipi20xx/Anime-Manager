# 订阅智能添加技能

## 触发条件

当用户说「帮我订阅」「订阅」「添加订阅」等关键词时触发此技能。

## 执行流程

### 步骤 1: 搜索作品

调用 `search_tmdb` 工具搜索，**参数必须保持用户输入的原样，不要修改**：

```
调用: search_tmdb
参数:
  - query: 用户提供的作品名称（保持原样！）
```

**重要**：不要修改用户的搜索词！如果用户说「又被杀掉了呢，侦探大人」，就原样传入这个字符串。

### 步骤 2: 展示搜索结果

从搜索结果中列出前 3-5 个匹配项，格式如下：

```
找到以下匹配作品：
1. [标题] (年份) - TMDB ID: xxx, 类型: tv/movie
2. [标题] (年份) - TMDB ID: xxx, 类型: tv/movie
```

### 步骤 3: 智能匹配或确认

- 如果用户明确指定了「第X季」「SX」「S2」等，从结果中找到对应季度的作品
- 如果只有一个高度匹配的结果，直接订阅
- 如果有多个可能的结果，询问用户要订阅哪一个

### 步骤 4: 执行订阅

使用 `add_subscription` 工具添加订阅：

```
调用: add_subscription
参数:
  - title: 作品标题
  - tmdb_id: TMDB ID
  - media_type: "tv" 或 "movie"（从搜索结果中获取）
  - season: 季度号（如果是剧集，默认为1）
```

## 重要规则

1. **保持搜索词原样**：不要修改用户输入的搜索词
2. **使用多类型搜索**：search_tmdb 会同时搜索剧集和电影
3. **智能匹配季度**：用户说「第二季」时，在结果中找标题包含「S2」「第二季」「Season 2」的条目
4. **完整执行**：搜索到正确结果后，直接执行订阅，不要停下来问用户
5. **报告结果**：订阅成功后，告诉用户订阅了什么作品

## 示例对话

**用户**: 帮我订阅 孤独摇滚

**助手**:
1. 调用 `search_tmdb("孤独摇滚")` - 注意：保持原样
2. 找到匹配结果：「孤独摇滚」(2022) - TMDB ID: 115036, 类型: tv
3. 调用 `add_subscription(title="孤独摇滚", tmdb_id="115036", media_type="tv", season=1)`
4. 回复：「✅ 已成功订阅《孤独摇滚》第1季」

**用户**: 订阅 又被杀掉了呢，侦探大人

**助手**:
1. 调用 `search_tmdb("又被杀掉了呢，侦探大人")` - 注意：保持原样！
2. 找到匹配结果
3. 调用 `add_subscription` 订阅
4. 回复：「✅ 已成功订阅」
