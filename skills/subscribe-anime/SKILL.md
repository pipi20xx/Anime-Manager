# 订阅智能添加技能

## 触发条件

当用户说「帮我订阅」「订阅」「添加订阅」「我想看」等关键词时触发此技能。

## 执行流程

### 步骤 1: 搜索作品

调用 `search_tmdb` 工具搜索，**参数必须保持用户输入的原样**：

```
调用: search_tmdb
参数:
  - query: 用户提供的作品名称（保持原样！）
```

**重要**：不要修改用户的搜索词！

### 步骤 2: 处理搜索结果

**情况 A：找到唯一高度匹配的结果（标题完全匹配或相似度 > 90%）**
- 直接调用 `add_subscription` 订阅
- 告诉用户订阅成功

**情况 B：找到多个可能的结果**
- 列出前 5 个匹配项，让用户选择
- 格式：
```
🔍 找到多个匹配结果：

| 编号 | 标题 | 年份 | 类型 | 评分 |
|:----:|:-----|:----:|:----:|:----:|
| 1 | 孤独摇滚 | 2022 | 剧集 | 8.5 |
| 2 | 孤独摇滚 特别篇 | 2023 | 剧集 | - |
| ...

请输入编号选择要订阅的作品：
```

**情况 C：未找到结果**
- 告诉用户未找到，建议换关键词

### 步骤 3: 执行订阅

用户选择后，调用 `add_subscription`：

```
调用: add_subscription
参数:
  - title: 作品标题
  - tmdb_id: TMDB ID（字符串）
  - media_type: "tv" 或 "movie"
  - season: 季度号（默认1）
```

## 重要规则

1. **保持搜索词原样**：不要修改用户输入
2. **智能判断**：唯一匹配时直接订阅，多个结果时让用户选择
3. **一次完成**：不要分多次询问，尽量一轮完成
4. **季度处理**：用户说「第二季」「S2」时，优先选择对应季度

## 示例

**用户**: 帮我订阅 孤独摇滚

**助手**:
1. 调用 `search_tmdb(query="孤独摇滚")`
2. 找到唯一匹配：「孤独摇滚」(2022) - TMDB ID: 115036
3. 调用 `add_subscription(title="孤独摇滚", tmdb_id="115036", media_type="tv", season=1)`
4. 回复：「✅ 已成功订阅《孤独摇滚》第1季」

---

**用户**: 订阅 柯南

**助手**:
1. 调用 `search_tmdb(query="柯南")`
2. 找到多个结果，列出表格
3. 等待用户选择编号
4. 用户输入 "1"
5. 调用 `add_subscription` 订阅第1个结果
6. 回复：「✅ 已成功订阅《名侦探柯南》」
