# 整理失败分析

当用户询问"为什么整理失败"、"帮我重试整理失败的条目"时使用此技能。

## 功能

1. **分析失败原因** - 检查文件路径、权限、目标目录等
2. **自动重试** - 尝试重新整理
3. **提供建议** - 给出用户可操作的建议

## 常见失败原因

| 原因类型 | 特征 | 解决方案 |
|----------|------|----------|
| 文件不存在 | 源文件被移动或删除 | 检查文件路径 |
| 权限不足 | 无法读取或写入文件 | 调整文件权限 |
| 目标目录不存在 | 整理路径配置错误 | 创建目录或修改配置 |
| 重命名冲突 | 目标文件已存在 | 启用覆盖选项 |

## API 端点

| 端点 | 用途 |
|------|------|
| `GET /api/organize/tasks` | 获取整理任务 |
| `POST /api/organize/retry` | 重试整理 |
| `GET /api/organize/history` | 整理历史 |
